| No | Judul Masalah | Deskripsi Masalah | Pendekatan Solusi yang Direkomendasikan |
|----|--------------|-------------------|-----------------------------------------|
| 1 | Suara pecah dan distorsi di volume tinggi | Speaker mengalami distorsi saat volume dinaikkan. Suara tidak bersih dan pecah. | Cek power rating speaker vs output amplifier; pertimbangkan upgrade speaker atau turunkan gain amplifier |
| 2 | Bass kurang bertenaga | Suara bass terasa tipis dan tidak menggetarkan; customer ingin bass lebih powerful untuk EDM, hip-hop, reggae. | Rekomendasikan subwoofer dedicated + amplifier mono; atur low-pass filter |
| 3 | Speaker bawaan mobil jelek, mau ganti | Speaker original pabrik terasa flat dan tidak bertenaga. Customer ingin upgrade langsung terasa. | Mulai dari speaker coaxial 6.5 inch; pertimbangkan component untuk kualitas lebih baik |
| 4 | Vocal dan mid range kurang jelas | Suara vocal penyanyi tidak terdengar jelas, tertutup bass atau musik. Mid range terasa tipis. | Upgrade ke speaker component dengan tweeter terpisah; pertimbangkan DSP untuk tuning EQ |
| 5 | Head Unit tidak bisa connect Bluetooth | Bluetooth di Head Unit tidak terdeteksi oleh HP atau sering putus-putus. | Reset pairing Bluetooth; update firmware head unit; jika masih gagal pertimbangkan penggantian head unit |
| 6 | Soundstage sempit, suara terasa datar | Posisi suara terasa menyatu dan tidak ada kedalaman. Tidak terasa seperti live performance. | Pasang tweeter di posisi A-pillar; gunakan speaker component; aktifkan time alignment jika tersedia |
| 7 | Ingin upgrade audio tapi budget terbatas | Customer mau upgrade sistem audio tapi perlu dilakukan bertahap sesuai budget. | Prioritas: (1) speaker pintu, (2) head unit DSP, (3) subwoofer aktif kolong, (4) amplifier |
| 8 | Ingin audio lebih keras untuk kompetisi/sound pressure | Customer ingin sistem audio sangat keras untuk kompetisi SPL atau sound pressure. | Sistem SPL: amplifier mono berdaya besar + subwoofer berimpedansi rendah; pertimbangkan boks ported |

## Lampiran H — Skema Database SQL (DDL Tabel Utama)

**File sumber:** `Database_Schemas_Newest.sql`  
**Database:** PostgreSQL 17.8 + pgvector (NeonDB)  
**Skema:** `sales`

Berikut adalah DDL (*Data Definition Language*) untuk tiga tabel utama sistem AudioMatch. DDL lengkap termasuk seluruh fungsi SQL tersedia di file `Database_Schemas_Newest.sql`.

```sql
-- Ekstensi yang diperlukan
CREATE EXTENSION IF NOT EXISTS vector;

-- ─────────────────────────────────────────────────────────────
-- Tabel 1: master_customer_problems
-- Basis pengetahuan masalah pelanggan (knowledge chunks)
-- ─────────────────────────────────────────────────────────────
CREATE TABLE IF NOT EXISTS sales.master_customer_problems (
    mcp_id                   UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    mcp_problem_title        TEXT NOT NULL,
    mcp_keywords             TEXT[],
    mcp_description          TEXT,
    mcp_recommended_approach TEXT,
    mcp_embedding            VECTOR(1024),
    mcp_search_vector        TSVECTOR GENERATED ALWAYS AS (
        to_tsvector('indonesian',
            COALESCE(mcp_problem_title, '') || ' ' ||
            COALESCE(mcp_description, '') || ' ' ||
            COALESCE(array_to_string(mcp_keywords, ' '), '')
        )
    ) STORED,
    mcp_is_active            BOOLEAN DEFAULT TRUE,
    mcp_created_at           TIMESTAMPTZ DEFAULT NOW(),
    mcp_updated_at           TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_problems_active
    ON sales.master_customer_problems (mcp_is_active);
CREATE INDEX IF NOT EXISTS idx_problems_embedding
    ON sales.master_customer_problems
    USING ivfflat (mcp_embedding vector_cosine_ops) WITH (lists = 10);
CREATE INDEX IF NOT EXISTS idx_problems_fts_title_desc
    ON sales.master_customer_problems USING gin (mcp_search_vector);


-- ─────────────────────────────────────────────────────────────
-- Tabel 2: master_products
-- Katalog produk audio mobil (111 produk)
-- ─────────────────────────────────────────────────────────────
CREATE TABLE IF NOT EXISTS sales.master_products (
    mp_id                    UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    mp_name                  TEXT NOT NULL,
    mp_category              TEXT NOT NULL,
    mp_brand                 TEXT NOT NULL,
    mp_price                 NUMERIC(12, 2),
    mp_description           TEXT,
    mp_image                 TEXT,
    mp_compatible_car_types  TEXT[],
    mp_recommended_car_sizes TEXT[],
    mp_solves_problem_id     UUID REFERENCES sales.master_customer_problems(mcp_id),
    mp_embedding             VECTOR(1024),
    mp_search_vector         TSVECTOR GENERATED ALWAYS AS (
        to_tsvector('indonesian',
            COALESCE(mp_name, '') || ' ' ||
            COALESCE(mp_brand, '') || ' ' ||
            COALESCE(mp_category, '') || ' ' ||
            COALESCE(mp_description, '')
        )
    ) STORED,
    mp_is_active             BOOLEAN DEFAULT TRUE,
    mp_created_at            TIMESTAMPTZ DEFAULT NOW(),
    mp_updated_at            TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_products_active
    ON sales.master_products (mp_is_active);
CREATE INDEX IF NOT EXISTS idx_products_category
    ON sales.master_products (mp_category);
CREATE INDEX IF NOT EXISTS idx_products_problem_fk
    ON sales.master_products (mp_solves_problem_id);
CREATE INDEX IF NOT EXISTS idx_products_fts_name_desc
    ON sales.master_products USING gin (mp_search_vector);


-- ─────────────────────────────────────────────────────────────
-- Tabel 3: master_cars
-- Spesifikasi audio bawaan kendaraan (230+ model)
-- ─────────────────────────────────────────────────────────────
CREATE TABLE IF NOT EXISTS sales.master_cars (
    mc_id                    UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    mc_brand                 TEXT NOT NULL,
    mc_model                 TEXT NOT NULL,
    mc_type                  TEXT,
    mc_size_category         TEXT,
    mc_dashboard_type        TEXT,
    mc_door_count            INTEGER,
    mc_factory_speaker_size  TEXT,
    mc_factory_speaker_count INTEGER,
    mc_special_notes         TEXT,
    mc_is_active             BOOLEAN DEFAULT TRUE,
    mc_created_at            TIMESTAMPTZ DEFAULT NOW(),
    mc_updated_at            TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_cars_active
    ON sales.master_cars (mc_is_active);
CREATE INDEX IF NOT EXISTS idx_cars_brand
    ON sales.master_cars (mc_brand);
CREATE INDEX IF NOT EXISTS idx_cars_model
    ON sales.master_cars (mc_model);
CREATE INDEX IF NOT EXISTS idx_cars_brand_model
    ON sales.master_cars (mc_brand, mc_model);
CREATE INDEX IF NOT EXISTS idx_cars_type
    ON sales.master_cars (mc_type);
```
